# Profile Genius - CLI Demo

![Demo](https://raw.githubusercontent.com/q8999513-ux/my-awesome-project/main/demo.gif)

## Installation

```bash
npm install -g profile-genius
```

## Usage

```bash
profile-genius
```

## Demo Steps

1. **Select Template** - Choose from 5 beautiful templates
2. **Fill Details** - Enter your information
3. **Preview** - See your generated README
4. **Export** - Copy to clipboard or save to file

## Available Templates

1. 👨‍💻 Developer - Full-featured developer profile
2. ✨ Minimalist - Clean and simple
3. 🧑‍💻 Hacker - Terminal-style aesthetic
4. 📝 Markdown Lover - Table-based layout
5. 🐱 Cat Lover - Fun cat-themed template
